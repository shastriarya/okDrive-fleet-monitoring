# AI Fleet Monitoring Dashboard

This project simulates a real-time fleet monitoring system for dashcam analytics.

## Features

- Real-time driver event streaming using WebSocket
- Speed violation detection
- Risk score calculation
- Dashboard analytics (Bar Chart + Pie Chart)
- Live dashcam video simulation
- Real-time alerts for speeding drivers

## Tech Stack

Frontend:
- React (Vite)
- Recharts
- Socket.io Client

Backend:
- Node.js
- Express
- Socket.io

Database:
- MySQL

## Installation

### Backend

cd fleet-backend
npm install
node server.js

### Frontend

cd fleet-frontend
npm install
npm run dev
